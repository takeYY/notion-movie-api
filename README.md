# notion-movie-api
観た映画のログ

## 初回設定
1. 環境変数の設定

    terminal
    ```bash
    touch app.env
    ```
    app.env
    ```
    NOTION_TOKEN="YOUR TOKEN"
    SECRET_KEY=b"YOUR SECRET KEY"
    DB_ID="YOUR DATABASE ID"
    TMDB_KEY="YOUR TMDB KEY"
    ```
2. jsonフォルダの作成
    terminal
    ```bash
    mkdir json
    ```
